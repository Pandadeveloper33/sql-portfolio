Welcome to my SQL portfolio! This repository showcases hands-on SQL projects focused on data analytics, fraud detection, and risk analysis. Each project simulates real-world datasets and demonstrates SQL querying skills to solve business problems.

## 📁 Projects

### 1. 🏦 Transaction Monitoring – Detecting Suspicious Activity
**Objective**: Simulate a basic banking system and write SQL queries to detect high-risk patterns.

**Tables**:
- `customers`: Customer details
- `accounts`: Account types linked to customers
- `transactions`: Deposits and withdrawals with timestamps

**Key SQL Skills Demonstrated**:
- Filtering with `WHERE`
- Sorting with `ORDER BY`
- Using `GROUP BY` and aggregate functions
- Basic anomaly detection with SQL logic
